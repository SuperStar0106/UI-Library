# Breakpoints

We use [breakpoints from xstyled](https://www.smooth-code.com/open-source/xstyled/docs/responsive/).

## Sizes

Default size from core theme. Get values from [this article](https://medium.com/@uiuxlab/the-most-used-responsive-breakpoints-in-2017-of-mine-9588e9bd3a8a).

```js
theme.breakpoints = {
  xs: 0,
  sm: 480,
  md: 736,
  lg: 980,
  xl: 1280
}
```
