# Themes

Here are all the possible values for your theme. These will be merged with the default theme which you can find here:

<Playground>
  {() => {
    const theme = createTheme()

    const ThemeValues = () => (
      <>
        <pre style={{
          fontFamily: 'monospace',
          fontSize: '14px'
        }}>{JSON.stringify(theme, null, 2)}</pre>
      </>
    )

    return <ThemeValues />

}}

</Playground>
